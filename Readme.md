# 设备管理系统 (EMS)

一个基于 Flask 框架开发的设备管理系统，支持虚拟机和物理机的管理、状态检查和数据导入等功能。

## 功能特性

- **设备管理**：支持添加、编辑、删除虚拟机和物理机设备
- **Excel 导入**：支持通过 Excel 文件批量导入设备数据
- **设备状态检查**：支持 ping 检查和 SSH 连接，获取 CPU、内存、磁盘使用率
- **用户认证**：基于角色的用户登录系统
- **响应式设计**：适配不同屏幕尺寸的设备
- **性能优化**：使用原生 JavaScript 替代 jQuery，减少加载时间
- **本地样式**：使用本地 CSS 文件，减少外部依赖

## 技术栈

- **后端**：Flask (Python)
- **数据库**：SQLite
- **前端**：HTML5, CSS3, 原生 JavaScript
- **数据可视化**：自定义表格和模态框
- **网络工具**：ping 命令、SSH 连接

## 快速开始

### 环境要求

- Python 3.6+
- pip 包管理器

### 安装步骤

1. **克隆或下载项目**

2. **安装依赖**

   在项目根目录下运行：
   
   ```bash
   pip install flask openpyxl paramiko
   ```

### 运行项目

1. **启动 Flask 服务器**

   在项目根目录下运行：
   
   ```bash
   python app.py
   ```

2. **访问系统**

   打开浏览器，访问：
   
   ```
   http://localhost:5000
   ```

3. **登录系统**

   使用默认账号登录（具体账号密码请参考数据库配置）：
   - 用户名：admin
   - 密码：admin123
   - 用户名：user
   - 密码：user123

## 使用说明

### 1. 虚拟机管理

- **添加设备**：点击"添加设备"按钮，填写设备信息并保存
- **编辑设备**：在设备列表中点击"修改"按钮，修改设备信息并保存
- **删除设备**：在设备列表中点击"删除"按钮，确认后删除设备
- **导入设备**：点击"导入Excel"按钮，上传包含设备信息的 Excel 文件
- **查看设备**：点击"查看设备"按钮，根据条件查询设备

### 2. 物理机管理

- **添加设备**：点击"添加设备"按钮，填写设备信息并保存
- **编辑设备**：在设备列表中点击"修改"按钮，修改设备信息并保存
- **删除设备**：在设备列表中点击"删除"按钮，确认后删除设备
- **导入设备**：点击"导入Excel"按钮，上传包含设备信息的 Excel 文件
- **查看设备**：点击"查看设备"按钮，根据条件查询设备

### 3. 设备状态检查

- **检查设备状态**：点击"检查设备状态"按钮，系统会自动检查所有设备的连接状态
- **查看检查结果**：检查完成后，结果会显示在表格中，包括连接状态、延迟、CPU/内存/磁盘使用率
- **高占用率警告**：当 CPU、内存或磁盘使用率超过 80% 时，会显示为红色加粗字体

### 4. 系统说明

- 点击导航栏中的"系统说明"链接，查看系统的功能介绍和使用指南

## 数据库结构

### users 表
| 字段名 | 数据类型 | 描述 |
|--------|---------|------|
| id | INTEGER | 用户 ID |
| username | TEXT | 用户名 |
| password | TEXT | 密码 |
| role_id | INTEGER | 角色 ID |

### roles 表
| 字段名 | 数据类型 | 描述 |
|--------|---------|------|
| id | INTEGER | 角色 ID |
| name | TEXT | 角色名称 |

## 常见问题

### 1. 数据库连接失败

- 确保 `data` 目录存在
- 确保 `device_management.db` 文件存在且有正确的权限

### 2. 设备状态检查失败

- 确保设备的 IP 地址正确
- 确保设备可以正常 ping 通
- 确保提供的 SSH 账号密码正确
- 确保设备的 SSH 服务正在运行

### 3. Excel 导入失败

- 确保 Excel 文件格式正确
- 确保表头与要求的格式一致
- 确保数据类型正确
